About

CSV2DAT is a series of online radio editing tools capable of creating .dat & .data files. These files can be loaded into the original CPS (computer programming software) and programed to radios.

How to Use

To create a radio program navigate to 'Online Editor' on the main menu. Many values can directly be edited with the online editor. Beware that navigating away from the editor page may erase any changes made.

Channels can be added or removed from the editor or they can be loaded from Chirp csv files. CSV files generated from any other program will likely not work. All none channel data will not be affected by importing csv files

To program your radio click on the 'Create .dat' button and a dat file will be downloaded to the download folder. Open this file with the original cps software and load it to your radio the standard way.
